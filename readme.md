A web app to play wordle as many times as you'd like. No daily limit.

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed on your machine.  

Installation:

- `npm install`   

To Run Locally:

- `cd wordle-clone`
- `npm start`  

To Visit the locally running App:

- `localhost:3000`
